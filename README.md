# PostgreSQL Assignment App

## Files
- config.yaml — DB host/port/database/user (password is entered at runtime)
- main.py — app that asks for credentials, connects and runs SELECT VERSION()
- requirements.txt — Python dependencies

## Run (quick)
1. Create venv (optional): python3 -m venv .venv
2. Activate: source .venv/bin/activate
3. Install deps: pip install -r requirements.txt
4. Run: python3 main.py
